Trabalhando somente com HTML5.

Primeiramente devemos fazer uso de um editor ou uma IDE
(que comporte linguagem HTML5) e um navegador.

  1. <!DOCTYPE HTML html>= Define que este documento é HTML5
  2. <html lang="pt-br"> = elemento raiz de uma página HTML e idioma do documento.
  3. <head> = elemento contém metainformações sobre a página HTML
  4. <title> = elemento especifica um título para a página HTML
    (que é mostrado na barra de título do navegador ou na guia da página)
  5. <body> = elemento define o corpo do documento e é um recipiente
    para todos os conteúdos visíveis, como cabeçalhos, parágrafos, imagens,
    hiperlinks, tabelas, listas, etc.
  6. <header> = elemento define área de cabeçalho.
  7. <nav> = elemento define área de navegação de links.
  8. <section> = elemento define uma seção do documento
  9. <article> = Define um conteúdo independente
  10. <aside> = Define o conteúdo além do conteúdo (como uma barra lateral)
  11. <footer> = Define um rodapé para o documento ou uma seção.
  12. <h1> = elemento define um título Principal
  13. <p> = elemento define o parágrafo

Criando Repositório GitHub.

Baixar o Git (https//git-scm.com), Download da ultima versão(64bit devido ao GitHubDesktop),
agora vamos buscar o GitHubDesktop(https://desktop.github.com), nos facilita a vida.Baixar para Windows(64bit). Instalamos na sequência 1° o Git /Next/Next até a hora de escolher o editor
Vamos usar o VSCode com Editor Padrão(Não use o inseder) /Next /Next / Next/ Next/
Install, neste momento está se criando repositório local. Não inicie o programa.
De um Boothing na maquina. Instalar o GitHubDesktop, é rápido clique em Sign in GitHub.com.
Para isso temos que ter uma conta no GitHub.com para podermos ter um repositório remoto. Cliando uma conta no GitHub.com, acessamos pelo navegador https://github.com , a primeira vêz clicar em
em sign up, Crie sua conta com um nome de usuário, um endereço de email válido, e um Password
(as instruções estão abaixo) Verifique a conta (um texte) criar conta, vai perguntar que tipo de
usuário você é. Para que vai usar, qual o interesse. Recebe um email de confirmação. Complete a ficha com uma foto, nome e outros dados. Voltamos para o GitHubDesktop no Username or email address:Colocamos o nome cadastrado na nossa conta, e usamos o nosso Password cadastrado.
Já está tudo configurado.

Gerenciando repositório GitHub

Clicamos no GitHubDesktop, ele já faz uma atualização do repositório, localiza alterações
.Clicamos em abrir o VSCode o mesmo é aberto e já localizamos as alterações e em que arquivo foi
modificado (M), retornamos ao GitHubDesktop onde podemos ver onde houve a mudança e
qual foi. E para atualizar o nosso repositório local temos que fazer um Commit to master,
temos que colocar um título para o Commit e Clicar em Commit to master já fazemos a atualização
para o Repositório local. E para atualizar o GitHub devemos Clicar em Push origen,
após isto o nosso Repositório do GitHub estará Ok.
